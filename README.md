## English

**DualMice** is a Windows utility that allows you to control a web browser with a **second mouse**, completely independent of your primary Windows cursor. This is specifically designed for users who want to browse the web on a secondary monitor while playing a full-screen game, without ever alt-tabbing or losing focus.

### ✨ Features
- **Independent Virtual Cursor:** Move your second mouse to control a virtual cursor inside a Google Chrome tab.
- **Full-Screen Gaming Support:** Your primary Windows cursor stays locked in your game.
- **Click & Scroll:** Emulates standard left/right clicks and scrolling inside the browser.
- **Keyboard Forwarding:** Type into the browser without losing game focus using "Keyboard Mode".
- **Customizable Appearance:** Adjust sensitivity, cursor size, opacity, and even use custom cursor images (`.png`, `.gif`, `.svg`).

### 🚀 Setup Instructions

1. **Download the Release**
   Download the latest ZIP file from the [Releases](../../releases) page and extract it.
2. **Launch the Controller**
   Run the `DualMice 1.0.0.exe` included in the distribution folder. *(If you don't see it on the taskbar, check the system tray icon in the bottom right corner.)*
3. **Install the Chrome Extension**
   - Open Google Chrome and navigate to `chrome://extensions/`.
   - Enable **Developer mode** in the top right corner.
   - Click **Load unpacked** and select the `Extension` folder provided in the downloaded package.
4. **Register Mouse 2**
   - While the app is open, move your second mouse.
   - The device will appear in the "Detected Devices" list at the bottom left.
   - Click the **Lock as Mouse 2** button to bind it.
5. **Start Browsing**
   Move your second mouse to see the virtual cursor appear inside your Chrome window!

### 💡 Browser Toolbar Usage
When your virtual cursor enters the browser, a floating toolbar appears at the top. You can use it to navigate pages since you cannot interact with the native browser UI.
- **< / >**: Back / Forward (Side buttons on your mouse also work!)
- **+**: Open a new tab
- **X**: Close the current tab
- **< / > (Tabs)**: Switch between tabs
- **Reload icon**: Refresh the page
- **Keyboard icon**: Toggle Keyboard Mode ON/OFF

### ⚠️ Notes
- **Keyboard Mode Warning:** While Keyboard Mode is active, keystrokes will be forwarded to the browser. Disable it by clicking the keyboard icon again.
- **Chrome Warning:** Loading unpacked extensions may trigger a warning when you open Chrome. Just click "Cancel" to ignore it.

## 日本語

**DualMice** は、フルスクリーンゲームをプレイしながら「2台目のマウス」を使ってブラウザを独立して操作できるWindows用ソフトウェアです。メインのWindowsカーソルはゲーム内にロックさせたまま、別のモニターのブラウザを快適に閲覧できます。

### ✨ 主な機能
- **独立した仮想カーソル:** マウス2の入力をキャプチャし、Google Chrome内に仮想カーソルを描画します。
- **フルスクリーンゲーム対応:** ゲームの操作を一切妨害しません。
- **クリック＆スクロール**: ブラウザ内での左/右クリック、ホイールスクロールをエミュレート。
- **キーボード転送:** 「キーボードモード」を使えば、ゲームのフォーカスを失わずにブラウザで文字入力が可能です。
- **柔軟なカスタマイズ:** 仮想カーソルの感度、サイズ、透明度、オリジナル画像(`png`, `gif`, `svg`)の設定が可能。

### 🚀 セットアップ手順

1. **ダウンロード**
   [Releases](../../releases) ページから最新版のZIPファイルをダウンロードし、解凍します。
2. **Controllerの起動**
   フォルダ内の `DualMice 1.0.0.exe` を起動します。（※タスクバーに表示されない場合は画面右下のタスクトレイを確認してください）
3. **Chrome拡張機能のインストール**
   - Chromeを開き、アドレスバーに `chrome://extensions/` と入力して移動します。
   - 画面右上の **デベロッパーモード** をオンにします。
   - **パッケージ化されていない拡張機能を読み込む** をクリックし、同梱されている `Extension` フォルダを選択します。
4. **マウス2の登録**
   - ソフトを起動した状態で、2台目のマウスを動かします。
   - 画面左下の「検出済みデバイス」リストにそのマウスが表示されます。
   - **マウス2としてロック** ボタンを押して紐付けます。
5. **使用開始**
   マウス2を動かすと、Chrome上に仮想カーソルが現れ、操作可能になります！

### 💡 ツールバー（コントローラー）の使い方
仮想カーソルがブラウザ内に入ると、画面上部にツールバーが表示されます。（拡張機能の仕様上、ブラウザ標準の戻るボタン等を押せないためです）
- **← / →**: 戻る / 進む（マウスのサイドボタンでも操作可能です）
- **＋**: 新しいタブを開く
- **✕**: 現在のタブを閉じる
- **← / → (タブ切り替え)**: 左右のタブに移動する
- **↺**: ページを再読み込み
- **⌨**: キーボードモードのオン/オフ

### ⚠️ 注意事項
- **キーボードモードについて**: オンになっている間はキー入力がブラウザに転送されます。もう一度「⌨」ボタンを押して解除してください。
- **Chromeの警告**: デベロッパーモードで拡張機能を追加しているため、Chrome起動時に警告ポップアップが出ることがありますが「キャンセル」で無視して問題ありません。
