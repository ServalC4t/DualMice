## English

**DualMice** is a Windows utility that lets you control a web browser with a **second mouse**, completely independent of your primary Windows cursor. Designed for users who want to browse the web on a secondary monitor while playing a full-screen game — no alt-tabbing, no focus loss.

### ✨ Features
- **Independent Virtual Cursor:** Mouse 2 controls a virtual cursor inside Google Chrome, without touching the Windows cursor.
- **Full-Screen Gaming Support:** Your primary cursor stays locked in your game.
- **Click & Scroll:** Left/right click and scroll emulation inside the browser.
- **Keyboard Forwarding:** Type into the browser without losing game focus via "Keyboard Mode".
- **Custom Cursor Images:** Use any `.png`, `.gif`, or `.svg` as your virtual cursor.
- **Auto-Hide:** Virtual cursor and toolbar auto-hide after 3 seconds of Mouse 2 inactivity.
- **Multi-Monitor Support:** Works across multiple monitors including vertical setups.
- **Draggable Toolbar:** Move the floating toolbar with Mouse 1 or Mouse 2.
- **YouTube Shorts Support:** Scroll through Shorts with Mouse 2.

### 🚀 Setup Instructions

1. **Download the Release**
   Download the latest ZIP from the [Releases](../../releases) page and extract it.

2. **Launch the Controller**
   Run `DualMice 1.0.2.exe`. *(If it doesn't appear on the taskbar, check the system tray in the bottom right.)*

3. **Install the Chrome Extension**
   - Open Chrome and go to `chrome://extensions/`.
   - Enable **Developer mode** (top right).
   - Click **Load unpacked** and select the `Extension` folder from the extracted package.

4. **Register Mouse 2**
   - While the app is running, move your second mouse.
   - It will appear in the **Detected Devices** list on the left.
   - Click **Lock as Mouse 2** to bind it.

5. **Start Browsing**
   Move Mouse 2 into a Chrome window — the virtual cursor will appear!

### 💡 Browser Toolbar
When the virtual cursor enters the browser, a floating toolbar appears at the top.

| Button | Action |
|--------|--------|
| ← / → | Back / Forward (side buttons also work) |
| ＋ | Open new tab |
| ✕ | Close current tab |
| ← / → (tabs) | Switch tabs |
| ↺ | Reload page |
| ⌨ | Toggle Keyboard Mode |

### ⚙️ Settings
| Setting | Description |
|---------|-------------|
| Sensitivity | Virtual cursor movement speed |
| Cursor Size | Size of the virtual cursor |
| Opacity | Transparency of the virtual cursor |
| Cursor Color | Color of the virtual cursor |
| Cursor Image | Custom cursor (PNG/GIF/SVG) |
| Lock Cursor While Using M2 | Prevents Mouse 2 from moving the Windows cursor |
| Block Clicks/Scrolls | Prevents Mouse 2 input from affecting Mouse 1 *(may cause slight delay)* |
| Start Minimized at Startup | Auto-launch minimized to tray on Windows startup |

## 日本語

**DualMice** は、フルスクリーンゲームをプレイしながら「2台目のマウス」を使ってブラウザを独立して操作できるWindows用ソフトウェアです。メインのWindowsカーソルはゲーム内にロックさせたまま、別のモニターのブラウザを快適に閲覧できます。

### ✨ 主な機能
- **独立した仮想カーソル:** マウス2の入力をキャプチャし、Google Chrome内に仮想カーソルを描画します。
- **フルスクリーンゲーム対応:** ゲームの操作を一切妨害しません。
- **クリック＆スクロール:** ブラウザ内での左/右クリック、ホイールスクロールをエミュレート。
- **キーボード転送:** 「キーボードモード」でゲームのフォーカスを失わずにブラウザで文字入力が可能。
- **カスタムカーソル画像:** `.png`、`.gif`、`.svg` をカーソルとして使用可能。
- **自動非表示:** マウス2が3秒間動かないと仮想カーソルとツールバーが自動で非表示になります。
- **マルチモニター対応:** 縦置きモニターを含む複数モニター構成に対応。
- **ドラッグ可能なツールバー:** マウス1・マウス2どちらでもツールバーを移動できます。
- **YouTube Shorts対応:** マウス2のスクロールでShortsをスクロール可能。

### 🚀 セットアップ手順

1. **ダウンロード**
   [Releases](../../releases) ページから最新版のZIPをダウンロードし、解凍します。

2. **Controllerの起動**
   `DualMice 1.0.2.exe` を起動します。（※タスクバーに表示されない場合は右下のタスクトレイを確認してください）

3. **Chrome拡張機能のインストール**
   - Chromeで `chrome://extensions/` を開きます。
   - 右上の **デベロッパーモード** をオンにします。
   - **パッケージ化されていない拡張機能を読み込む** をクリックし、`Extension` フォルダを選択します。

4. **マウス2の登録**
   - ソフトを起動した状態でマウス2を動かします。
   - 左側の「検出済みデバイス」リストに表示されます。
   - **マウス2としてロック** ボタンを押して紐付けます。

5. **使用開始**
   マウス2をChromeウィンドウ内に移動すると仮想カーソルが表示されます！

### 💡 ツールバー（コントローラー）
仮想カーソルがブラウザ内に入ると、画面上部にツールバーが表示されます。

| ボタン | 操作 |
|--------|------|
| ← / → | 戻る / 進む（サイドボタンでも操作可能） |
| ＋ | 新しいタブを開く |
| ✕ | 現在のタブを閉じる |
| ← / → (タブ) | タブを切り替える |
| ↺ | ページを再読み込み |
| ⌨ | キーボードモードのオン/オフ |

### ⚙️ 設定項目
| 設定 | 説明 |
|------|------|
| 感度 | 仮想カーソルの移動速度 |
| カーソルサイズ | 仮想カーソルの大きさ |
| 不透明度 | 仮想カーソルの透明度 |
| カーソルの色 | 仮想カーソルの色 |
| カーソル画像 | カスタム画像（PNG/GIF/SVG） |
| マウス２を操作中にカーソルをロックする | マウス2がWindowsカーソルを動かさないよう補正 |
| クリック/スクロールをブロック | マウス2の入力がマウス1に影響しないようにする *(遅延が生じる場合あり)* |
| Windows起動時に最小化して開始 | PC起動時にトレイに最小化して自動起動 |
